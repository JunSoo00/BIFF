# 📌 Project : BIFF clone (부산국제영화제 2024)
2024년 부산국제영화제 웹 사이트의 콘텐츠 전시 방식, Scroll Interaction 등을 학습하고 구현하기 위해 제작된 클론코딩 프로젝트입니다.

메인 이미지의 `Swiper 커스터마이징`과 JavaScript를 활용한 `loop 컨텐츠`를 학습하고 활용할 수 있는 기회였습니다.

상업적 목적이 없으며 학습 및 개인 포트폴리오용입니다.

<br>

## 🖥️ Live Demo
🔗 배포 링크 : https://junsoo00.github.io/BIFF/

<br>

## ✨ Features
- 메인페이지 레이아웃 구현
- Swiper 커스터마이징, Header영역 및 버튼들에 hover기반 인터렉션 적용
- JavaScript를 활용한 이미지 삽입 및 loop 컨텐츠 삽입

<br>

## 📒 Development Notes
### 1. Main page
- 메인 이미지 Swiper를 커스터마이징 -> fade 이펙트를 적용하여 BIFF 특유의 '시네마틱'한 분위기를 표현하였습니다.
- Header 영역에 hover시 노출되는 서브 메뉴에 opacity + transition을 활용하여
- 각 버튼에는 scale, opacity 기반 인터렉션을 적용하여 사용자 행동(hover)에 대한 시각적 피드백을 강화하였습니다.
- 'Inside BIFF', 'Video', 'Archive' 항목에 JavaScript로 콘텐츠 배열을 구성하고, Loop 옵션을 활용하여 콘텐츠가 끊기지 않고 연속적으로 전시되도록 구현하였습니다.
